Syarat dan ketentuan Penggunaan

Diperbarui: 18 Januari 2024

Selamat datang di situs web PT Gemilang Energi Teknologi (https://www.gemilangenergiteknologi.com).

Harap membaca dengan seksama syarat dan ketentuan ini sebelum menggunakan situs web kami. Dengan mengakses atau menggunakan situs web ini, Anda setuju untuk terikat oleh syarat dan ketentuan berikut

1. Penggunaan Situs Web

Anda setuju untuk menggunakan situs web ini sesuai dengan semua undang-undang yang berlaku dan sesuai dengan syarat dan ketentuan ini.

Kami tidak mengizinkan penggunaan situs web ini untuk tujuan ilegal.

2. Hak Kekayaan Intelektual

Seluruh materi di situs web ini, termasuk tetapi tidak terbatas pada teks, gambar, desain, dan logo, dilindungi oleh hak cipta, merek dagang, dan hak kekayaan intelektual lainnya yang dimiliki oleh PT Gemilang Energi Teknologi, kecuali dinyatakan berbeda pada keterangan yang diberikan di bawah gambar atau bagian lainnya,

Anda dilarang untuk menggunakan kembali atau mendistribusikan materi dari situs web ini tanpa izin tertulis dari PT Gemilang Energi Teknologi.

3. Tautan ke Situs Web Pihak Ketiga

Situs web kami mungkin berisi tautan ke situs web pihak ketiga. PT Gemilang Energi Teknologi tidak bertanggung jawab atas konten atau kebijakan privasi situs web pihak ketiga tersebut.

4. Perubahan pada Syarat dan Ketentuan

PT Gemilang Energi Teknologi berhak untuk mengubah syarat dan ketentuan ini kapan saja tanpa pemberitahuan sebelumnya. Perubahan tersebut akan diberlakukan segera setelah diperbarui di situs web kami.

5. Penyelesaian Sengketa

Setiap sengketa yang timbul terkait dengan penggunaan situs web ini akan diselesaikan secara damai. Jika penyelesaian damai tidak mungkin, sengketa tersebut akan diselesaikan sesuai dengan hukum yang berlaku di yurisdiksi yang berlaku.

6. Hubungi Kami

Jika Anda memiliki pertanyaan atau kekhawatiran mengenai syarat dan ketentuan ini, silakan hubungi kami melalui gemilangenergieteknologi.get@gmail.com atau alamat kantor yang tercantum di situs web kami.
